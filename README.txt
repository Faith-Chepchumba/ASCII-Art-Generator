# Cute ASCII Art Generator

## Overview

A Ruby program that generates cute ASCII art, like hearts and stars.

## Features

- Choose between generating a heart or a star.
- Outputs the ASCII art to the terminal.

## Technologies Used

- Ruby

## How to Run

1. Run `ascii_art_generator.rb` using Ruby: `ruby ascii_art_generator.rb`.

2. Follow the prompt to select an art piece.
